## 🚀 How to Use New Features

### **Create a Poll:**
1. Click "Post" button
2. Click poll icon (bar chart)
3. Enter question & 2-4 options
4. Set duration (1 hour - 7 days)
5. Post!

### **Bookmark Posts:**
- Click bookmark icon on any post
- Access all bookmarks from sidebar

### **Create Lists:**
1. Go to Lists page
2. Click "New List"
3. Add name & description
4. Make public or private
5. Add users to your list

### **Enable Monetization:**
1. Go to Monetization page
2. Click "Enable Monetization"
3. Start earning from:
   - Video ads
   - Subscriptions
   - Tips
   - Products

### **Schedule Posts:**
- Compose post
- Click schedule icon
- Select date & time
- Post will publish automatically
